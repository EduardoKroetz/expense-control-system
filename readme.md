# Expense Control System

## Requisitos

* Node.js 18+
* Git
* .NET 10

## Executar o projeto

```bash
git clone https://github.com/EduardoKroetz/expense-control-system.git
cd expense-control-system/frontend
npm install
npm run dev
```

Depois abra no navegador:

```
http://localhost:5173
```

O backend e o banco SQLite são iniciados automaticamente.
